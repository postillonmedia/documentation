# react-native-theme

A HOC for theming your react-native app



## Installation

**Using yarn**

`$ yarn add @postillon/react-native-theme`

**Using npm**

`$ npm install @postillon/react-native-theme --save`



## Example

Try it on [Expo](https://snack.expo.io/@danielang/react-native-theme).
